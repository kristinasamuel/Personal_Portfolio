# Kristina Samuel - Personal Portfolio

A modern, responsive portfolio website showcasing my work as a Full Stack Developer and Agentic AI Developer, built with Next.js, Tailwind CSS, and Framer Motion.

## 🌟 Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Powered by Framer Motion for engaging user experience
- **Performance Optimized**: Fast loading with lazy loading and image optimization
- **SEO Friendly**: Built with best practices for search engine optimization

## 🛠️ Tech Stack

- **Frontend**: Next.js 16+, TypeScript, React 19
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion, Typewriter Effect
- **Deployment**: Vercel
